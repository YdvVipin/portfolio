Recommended local toolchain (run on your machine):

- ImageMagick or any image editor (optional) to convert SVG -> PNG posters
- (Optional) TTS/voiceover tool (ElevenLabs, local TTS) to create voiced narration

High-level edit flow (one-off commands to run for each demo):
1) Convert SVG poster to PNG (optional, ffmpeg or imagemagick)
   - Using ImageMagick:
     convert -density 150 poster.svg -resize 1280x720 poster.png
   - Using ffmpeg (if imagemagick not available):
     ffmpeg -y -i poster.svg -vf scale=1280:720 poster.png

2) (Optional) Create short intro/outro clips from a poster image
   - 3s intro (1280x720, 30fps):
     ffmpeg -y -loop 1 -i poster.png -c:v libx264 -t 3 -pix_fmt yuv420p -vf "format=yuv420p" intro.mp4
   - 5s outro (use another poster or same):
     ffmpeg -y -loop 1 -i poster.png -c:v libx264 -t 5 -pix_fmt yuv420p -vf "format=yuv420p" outro.mp4

3) Concatenate intro + original + outro into a single file (re-encode)
   - ffmpeg -y -i intro.mp4 -i input.mp4 -i outro.mp4 -filter_complex "[0:v][0:a][1:v][1:a][2:v][2:a]concat=n=3:v=1:a=1[outv][outa]" -map "[outv]" -map "[outa]" -c:v libx264 -crf 23 -preset medium -c:a aac -b:a 128k combined.mp4
   - If your intro/outro lack audio, replace their audio stream with silent one: (ffmpeg -f lavfi -i anullsrc=channel_layout=stereo:sample_rate=44100)

4) Burn-in subtitles from the prepared SRT file (optional)
   - ffmpeg -y -i combined.mp4 -vf "subtitles=assets/subs/playwright-bdd-framework-demo.srt:force_style='FontName=Inter,FontSize=20,PrimaryColour=&HFFFFFF&'" -c:a copy combined_subs.mp4
   - Note: ffmpeg's subtitles filter works well with .srt; if fonts are missing, it uses defaults.

5) Overlay lower-third captions / short callouts at specific times (example)
   - Example: add a lower-third text between 4s and 12s
     ffmpeg -y -i combined.mp4 -vf "drawtext=fontfile=/path/to/DejaVuSans-Bold.ttf:text='Brittle tests slow teams down':fontcolor=white:fontsize=28:box=1:boxcolor=black@0.5:boxborderw=8:x=40:y=640:enable='between(t,4,12)'" -c:a copy annotated.mp4

6) Normalize and compress final video
   - ffmpeg -y -i annotated.mp4 -c:v libx264 -crf 23 -preset medium -c:a aac -b:a 128k final_optimized.mp4

7) Optional: burn-in SRT and combine steps in one pipeline
   - ffmpeg -y -i intro.mp4 -i input.mp4 -i outro.mp4 -filter_complex "[0:v]scale=1280:720,setsar=1[v0];[1:v]scale=1280:720,setsar=1[v1];[2:v]scale=1280:720,setsar=1[v2];[v0][0:a][v1][1:a][v2][2:a]concat=n=3:v=1:a=1[outv][outa]" -map "[outv]" -map "[outa]" -vf "subtitles=assets/subs/playwright-bdd-framework-demo.srt" -c:v libx264 -crf 23 -preset medium -c:a aac final_with_subs.mp4

Notes & tips:
- If your machine lacks the exact fonts referenced, use a system font path in the drawtext command (DejaVuSans is common on Linux). Example font path: /usr/share/fonts/truetype/dejavu/DejaVuSans-Bold.ttf
- For best quality / reasonable file size, use CRF 20-24 (lower number -> higher quality). Adjust bitrate for audio as needed.
- If you want background music, prepare a music track (loopable, royalty-free), then mix it by adding -i music.mp3 and using -filter_complex to adjust volumes and apply audio ducking when voiceover is present.
