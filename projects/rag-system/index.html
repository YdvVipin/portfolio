Key Features

A domain-specific RAG pipeline built specifically for QA and testing knowledge.

Document Ingestion

Process PDFs, Markdown files, and scraped web content. Intelligent chunking and metadata extraction for optimal retrieval.

25+ Source Scrapers

Modular data collectors for Playwright docs, Selenium, Stack Overflow, GitHub repos, blogs, Wikipedia, and Arxiv papers.

FAISS Vector Store

Fast semantic search over embedded documents using sentence-transformers. Sub-second retrieval across the entire knowledge base.

Interactive Chat

Both Streamlit web UI and terminal interface for Q&A. Ask natural language questions about testing frameworks and best practices.

Data Validation

Quality checks, deduplication, and collection statistics. Ensures clean, high-quality training data for accurate responses.

Resume & Progress

Interrupt and resume long-running collection jobs. Progress tracking and logging for multi-hour scraping sessions.
