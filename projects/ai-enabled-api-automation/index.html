AI-Enabled API Automation Framework

Project Overview

A comprehensive API testing framework that leverages AI capabilities for intelligent test automation, featuring BDD (Behavior-Driven Development) approach with Gherkin syntax, performance testing, and detailed reporting.

Technologies Used

Python Behave (BDD) Requests Allure Reports JSON Schema Validation Performance Testing CI/CD

Key Features

  • BDD Framework: Behavior-driven development using Gherkin syntax for readable test scenarios
  • API Testing: Comprehensive testing of Pokemon API endpoints with validation
  • Performance Testing: Load testing and response time monitoring
  • Schema Validation: JSON schema validation for API responses
  • Allure Reporting: Rich HTML reports with detailed test results and metrics
  • Data-Driven Testing: Parameterized tests with external test data
  • Negative Testing: Error handling and edge case validation
