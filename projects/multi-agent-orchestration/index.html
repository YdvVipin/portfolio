Key Features

Browser Recording

Record browser interactions and convert them to test scripts. Supports clicks, inputs, navigation, and complex user flows.

MCP Protocol

Model Context Protocol for standardized agent communication. Each agent exposes tools and resources via JSON-RPC.

A2A Orchestration

Agent-to-Agent communication for complex multi-step QA workflows. Agents delegate tasks and share context.

Streamlit Dashboard

Real-time analytics dashboard with test metrics, agent status, coverage visualizations, and execution history.

Docker Deployment

Containerized microservice architecture with Docker Compose. PostgreSQL, FastAPI, and agent services.

Self-Healing Tests

AI-powered automatic repair of failing tests. Broken selectors, timing issues, and assertion mismatches fixed on the fly.
